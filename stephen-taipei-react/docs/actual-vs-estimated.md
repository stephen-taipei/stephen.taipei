### 1. Web Workers (750 → 0)
- **最大差異**：預估 750 項，實際 0 項
- **原因**：該主題頁面可能尚未建立或無可用項目

### 2. 其他主題 (~1000 → 300-800)
- **平均差異**：每個主題少 200-600 項
- **可能原因**：
  - 原始估計過於樂觀（假設每個主題都有 1000 項）
  - 實際項目數量取決於各 GitHub repo 的內容
  - 部分項目可能尚未整合到網站

### 3. Tailwind UI (1043 = 1043)
- **唯一準確**：這是唯一事先知道確切數量的主題

## 驗證方法

執行以下命令可重新驗證：
```bash
node get-actual-counts.js
```

1. 訪問每個主題頁面
